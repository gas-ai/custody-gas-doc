#### Request Parameters

<ParamField query="receive_address" type="string" required>
    receive address
</ParamField>
<ParamField query="address_to" type="string" required>
    The transfer address for estimating miner fees.
</ParamField>
<ParamField query="contract_address" type="string" required>
    Contract address for estimating miner fees.
</ParamField>
<ParamField query="service_charge_type" type="string" required>
    rent time code: Request the order/price interface to return the cost codes for
    minutes, hours, days, and different times.
    <p>eg:10010 - 10 minutes, 20001 - 1 hour, 30001 - 1 day</p>
</ParamField>
