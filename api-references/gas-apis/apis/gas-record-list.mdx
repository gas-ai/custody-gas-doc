---
title: 查询记录
api: GET /api/mpc/tron/gas/record/list
description: "通过id获取特定的代理记录"
---

#### 请求参数数据结构

<ParamField query="request_ids" type="string" required>
  多个request_id的字符串，英文逗号分割，例:123,345
</ParamField>
