---
title: Tron购买资源异步回调通知
---

`POST /客户回调通知地址`

#### I.请求参数数据结构

| Param      | 类型    | 是否必须 | 说明                                               |
| :--------- | :------ | :------- | :------------------------------------------------- |
| trade_no   | String  | 是       | 资源购买交易no                                     |
| request_id | String  | 是       | 唯一标识                                           |
| status     | Integer | 是       | 交易状态 0创建订单成功 1代理资源成功 2代理资源失败 10资源已回收 |

#### II.请求header
`Content-Type:application/x-www-form-urlencoded`

#### III. 交易通知响应参数统一格式

返回字符串 （注意此处返回参数**无需进行加密**）：
* `SUCCESS`表示成功
* `FAILURE`表示失败
