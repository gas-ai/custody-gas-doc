#### 请求参数数据结构

| Param      | 类型    | 是否必须 | 说明                                               |
| :--------- | :------ | :------- | :------------------------------------------------- |
| trade_no   | String  | 是       | 资源购买交易no                                     |
| request_id | String  | 是       | 唯一标识                                           |
| status     | Integer | 是       | 交易状态 0创建订单成功 1代理资源成功 2代理资源失败 |
